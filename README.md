# Black Griffon Player's Guide

Static HTML document for 'Black Griffon Player's Guide'.

Requirements:

  * [Node.js](http://nodejs.org/)

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To test:

    $ npm start

And open http://localhost:8000 in your browser.

To clean build:

    $ npm clean

To publish as a PDF:

    $ yarn publish
