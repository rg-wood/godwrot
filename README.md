# Grisly Eye Games Document Template

Static HTML document template for Grisly Eye Games documents.

Requirements:

  * [Node.js](http://nodejs.org/)

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To test:

    $ npm start

And open http://localhost:8000 in your browser.

To clean build:

    $ npm clean

To publish as a PDF:

    $ npm run publish
